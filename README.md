# 🚀 Smart Lead Dashboard

A modern full-stack Lead Management Dashboard built to streamline lead tracking, user authentication, and business workflow management. The application provides a secure and scalable system with advanced filtering, search capabilities, role-based access control, and a responsive user interface.

# ✨ Features

## 🔐 Authentication & Security
- User Signup & Login
- JWT-based Authentication
- Protected Routes
- Secure Password Hashing
- Role-Based Access Control (Admin & Sales User)

## 📊 Lead Management
- Create, Update, Delete Leads
- View Single Lead Details
- Lead Status Management
- Advanced Lead Search & Filtering
- Sorting by Latest & Oldest
- Backend Pagination
- CSV Export Functionality

## 🎨 User Experience
- Responsive Dashboard UI
- Dark Mode Support
- Reusable Components
- Form Validation
- Loading & Error States
- Modern UI with Lucide React Icons

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/adiba-anwar01/smart-leads.git
cd smart-leads
```
# 🔧 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

# 🔧 Backend Setup

```bash
cd backend
npm install
npm run dev
```

# 🔑 Environment Variables

Create a `.env` file inside the backend folder and add your environment variables.
