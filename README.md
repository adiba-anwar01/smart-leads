# 🚀 Smart Lead Dashboard

Smart Lead Dashboard is a full-stack web application for managing and tracking leads efficiently. It provides authentication, a user dashboard, and seamless communication between frontend and backend.

---

## 🌐 Live Demo

Frontend:  
https://smart-leads-oifv.vercel.app  

Backend:  
https://smart-leads-eiy6.onrender.com  

---

## 📌 Features

- User authentication (Signup / Login)
- Protected dashboard
- Lead management system
- REST API integration
- Responsive UI
- Secure JWT-based authentication

---

## 🛠️ Tech Stack

### Frontend:
- React (Vite)
- TypeScript
- Axios
- React Router

### Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
